# Adivina Quién

## Descripción general
Esta es una versión en consola del clásico juego Adivina Quién, desarrollada completamente en Python. El sistema carga una base de datos local y permite al usuario descubrir el personaje secreto mediante preguntas clave.

## Organización del Proyecto
* `main.py`: Punto de entrada que ejecuta el juego.
* `data/`: Contiene `personajes.txt` con las características y generará `historial.txt` al jugar.
* `modules/`: Código dividido en operaciones de archivos, lógica de juego, manejo de personajes y validación de preguntas.
* `docs/`: Documentación técnica.

## Cómo ejecutarlo
Ubicado en la raíz del proyecto, ejecuta el siguiente comando en tu terminal:
`python main.py`
